Passos rápidos

  1. Publique a pasta api na Vercel.
  2. Defina FINNHUB_API_KEY na Vercel.
  3. Copie a URL da Vercel para data/app-config.json.
  4. Publique esta pasta inteira no Surge.

Notas importantes

Farside funciona como fonte de fluxo diário por emissor.
BitcoinTreasuries funciona como breadth para companhias abertas com BTC.
Finnhub complementa com quotes, perfis e métricas de ETF.
Se algum provider falhar, o frontend continua carregando os demais módulos.
Projeto estático pronto para Surge. Backend proxy e agregador pronto para Vercel.
